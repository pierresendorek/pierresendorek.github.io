Java applets

This page presents some applets I created with Java. I hope it could catch someone’s attention and be the beginning for other projects.


2D Wave Equation on a rectangle (d'Alembert) - October 31, 2009
A java applet solving a smoothed version of d'Alembert discretized Wave equation. Waves can be generated by user with mouse.
Another version here with a thiner grid and planar waves.

Spring equation (Band Pass filtering) - August 1, 2009
A java applet which solves linear second order discretized differential equations*, the second member beeing entered in real time by the mouse. Possibilty to change the swiftness of the spring and the friction. Visualisation of the sonograms.
** :

Particles in a box - August 1, 2009
A java applet containing many bouncing particles attracting/repulsing each other. Control parameters : gravity, friction, wall bounce, equilibrium distance and another distance related parameter.

2x1D Wave equation on a rope (d'Alembert) - March 22, 2009
A java applet solving d'Alembert discretized equation independently on x and y axis. The result is a rope on which waves propagate. This is one of the first codes I wrote in Java so it is a little bit minimalist.
